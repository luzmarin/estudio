proceso

Primero de todo haremos una primera reunión para conocernos y saber mas de ese proyecto que tienes en mente.

Empezamos con una primera fase de investigación, y cuando ya lo tenemos todo empezamos con las propuestas de diseño.

Una vez que ya se tiene varias ideas de diseño, te las presentamos para que nos des feedback e ir avanzando con el proyecto.

Cuando ya se tiene todo el proyecto terminado se te entrega para que puedas empezar con esa idea de empresa.
